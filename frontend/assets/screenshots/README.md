# Play Store / Amazon App Store Screenshots

Bhai, yahan pe screenshots rakhiye Google Play Store ke liye:

## Required Sizes
- Phone: 16:9 (3200x1800 ya 2880x1440) ya 2:3 (1080x1920)
- Tablet: 4:3 (3200x2400 ya 2560x1920) ya 16:9 (2560x1440)
- **Android 12+ ke liye**: 16:9 (1920x1080) aur 20:9 (3840x2160) ya 21:9 ratio

## Minimum 8 screenshots chahiye per device:
1. **Home Screen** - Customer selection page
2. **Medicine List** - Customer dashboard
3. **Buy Screen** - Location + price breakdown
4. **Payment Screen** - Order confirmation
5. **My Orders** - Customer order history
6. **Shop Dashboard** - Dukandar view
7. **Order Details** - Map + live location
8. **Analytics** - Daily report + charts

## Bhai, screenshots tabhi kaafi hai jab:
- Hindi/Hinglish text saaf dikhe
- Real medicine names aur prices ho
- Dark/light mode mix rakhiye
- Different phone sizes (phone + tablet)

## Upload karna:
1. Play Store Console > Store presence > Main store listing
2. Screenshots section me drag-drop karein
3. Feature graphic: `assets/screenshots/feature-graphic.png` (1024x500)
4. Promo video: Optional, 30 sec max
